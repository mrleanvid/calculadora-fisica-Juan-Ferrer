## 📋 Descripción del Proyecto

**Calculadora de Magnitudes Físicas** es una aplicación web desarrollada con **HTML, CSS y JavaScript** para la materia *Construcción de Algoritmos* de la carrera *Ingeniería de Software*.

La aplicación implementa **10 operaciones físicas fundamentales** con una interfaz interactiva, visualmente atractiva y responsiva, usando una paleta de colores en tonos **blanco, morado, negro y azul**.

### 🔬 Fórmulas Implementadas

| # | Magnitud | Fórmula | Unidad |
|---|----------|---------|--------|
| 1 | Velocidad | `v = d / t` | m/s |
| 2 | Aceleración | `a = Δv / Δt` | m/s² |
| 3 | Fuerza | `F = m · a` | N (Newtons) |
| 4 | Trabajo | `W = F · d · cos(θ)` | J (Joules) |
| 5 | Energía Cinética | `K = ½ · m · v²` | J (Joules) |
| 6 | Energía Potencial Gravitatoria | `U = m · g · h` | J (Joules) |
| 7 | Densidad | `ρ = m / V` | kg/m³ |
| 8 | Presión | `P = F / A` | Pa (Pascales) |
| 9 | Carga Eléctrica | `q = I · t` | C (Coulombs) |
| 10 | Ley de Ohm | `V = I · R` | V (Voltios) |

### ✨ Características

- ✅ Interfaz interactiva y visualmente atractiva
- ✅ Validaciones en tiempo real (campos vacíos, división por cero, valores negativos)
- ✅ Historial de cálculos persistente (localStorage)
- ✅ Animación de partículas en el fondo
- ✅ Diseño responsivo (móvil, tablet, escritorio)
- ✅ Soporte de tecla Enter para calcular
- ✅ Tooltips informativos
- ✅ Valor de gravedad (g = 9.81) precargado
- ✅ Resultados con notación científica para valores extremos

### Método 2: Con servidor local (recomendado)

**Usando VS Code con Live Server:**
1. Abre la carpeta en Visual Studio Code
2. Instala la extensión **Live Server** (si no la tienes)
3. Haz clic derecho sobre `index.html` → **Open with Live Server**
4. La aplicación se abrirá en `http://127.0.0.1:5500`
